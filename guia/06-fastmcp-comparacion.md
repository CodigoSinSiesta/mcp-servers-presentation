# FastMCP vs SDK Oficial

## Tiempo estimado
3 minutos

## Puntos clave
- Comparación: ~100 líneas TS vs 6 líneas Python.
- FastMCP automatiza: list_tools, JSON Schema, dispatcher y TextContent.
- Type hints de Python se convierten en JSON Schema automáticamente.

## Datos relevantes
- Docstrings de las funciones se usan como descripciones para el LLM.
- Cuándo usar cada uno: FastMCP para prototipos, SDK para producción.
